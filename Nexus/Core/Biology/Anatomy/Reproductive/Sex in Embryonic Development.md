Urogenital system at first has Mullerian duct that can become the female reproductive system and Wolffian duct that can become the male version.
Y chromosome has SRY gene that, if present, activates Wolffian duct and so gives testicles and penis. If SRY is absent then we get female.

**Hermaphroditism** is. If SRY is absent then we get female.

**Hermaphroditism** is when some disbalance creates a person who partly has both
