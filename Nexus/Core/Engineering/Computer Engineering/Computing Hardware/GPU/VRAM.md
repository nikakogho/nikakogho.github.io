Video Random Access Memory. A specialized memory for [[GPU|GPUs]].

Unlike [[RAM]], VRAM is only for GPUs and is physically on the graphics card so close to GPU chip.
A GPU can still use normal RAM but it is much slower and typical use cases in both gaming and [[AI]] use VRAM.

The main 2 types are **GDDR (Graphics Double Data Rate)** and **HBM (High-Bandwidth Memory)**.
