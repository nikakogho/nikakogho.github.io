Browser Extensible Data.
For [[Genomic Feature Annotation]]
![IMG_5676.png](img_5676.png)

## Basic Format

1. Chromosome number
2. Start bp (0-based)
3. End bp

## Extended Format

1. Chromosome number
2. Start bp
3. End bp
4. Exon name
5. Score (for display like importance)
6. Strand (+ or -)
7. thick_start - section to draw thickly in display
8. thick_end
9. rgb (display color)
